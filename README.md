# EduLauncher
One of the First Minecraft Education Launcher

# Features
Manage Addons

Manage Worlds

Backup Worlds

Manage Backups

And a Little More!!!

# Requirements
Minecraft Education (Microsoft Store version only, sorry mac!)

A Working  Minecraft Education Account

Git installed (You use git bash to run it)

Python installed

And a valid internet connection!

# Reminder

This Is an early beta...

# Instructions

1. Download the ZIP file and extract it
2. Open the folder and right click
3. Then once right clicked select Open Git Bash Here
4. Then run "sh launch.sh"
5. A Window will open up!
